Recent Changes

New Features
  • Introduced animated 3D visualizations for the Flow Project Site, fuel lines, and starter building floors in the management grid.
  • Added a new status display showing flow points, population, and workforce with tiered icons and styles.
  • Implemented a new "Flow Project Site" building with custom rendering and under-construction messaging.
Improvements
  • Enhanced top bar controls with grouped buttons, separators, and confirmation dialogs for special actions.
  • Updated resource sidebar management to use global state for improved consistency.
  • Refined resource and population displays, including reordering and visual dividers.
Style
  • Updated UI styles for buttons, status display, and improved responsive layouts for better usability across devices.
Bug Fixes
  • Adjusted dome and building placements for improved grid initialization and consistency.
Chores
  • Removed legacy and backup code files to streamline codebase.

Soon

Currently working on balancing and 3D puzzle areas

On the Horizon

Once the management gameplay is properly balanced, the focus will shift to developing the 3D puzzle areas. These immersive environments will challenge players to solve interactive puzzles that represent the AI's learning process about the physical world, deepening the connection between the colony management and the embodiment narrative.

The puzzles will also unlock new capabilities for your colony, creating a meaningful progression loop between the management and puzzle gameplay elements.
